# vmlab.wcl

_file_

The lab definition, found by walking up from the current directory (like git).

The lab definition file. Lab-scoped verbs locate it by walking up from the current
directory to the nearest `vmlab.wcl`. It declares one `lab {}` and/or any number of
`template {}` blocks. Every file in the lab begins with `import <vmlab.wcl>`.
